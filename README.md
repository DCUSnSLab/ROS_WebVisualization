
#### INSTALL
react
rosblib
nodejs
npm

#### SETTING
1. create-react-app '프로젝트명'
2. git clone 'https://github.com/PPOS2EUN/ROS.git'

#### START
1. 상단에 |> Edit configurations ... 클릭
2. npm 추가
3. npm setting
    - name : start 
    - package.json : package.json 경로
    - command : run
    - Scripts : start
4. RUN |> 클릭
5. ws://localhost:3000 접속 확인
